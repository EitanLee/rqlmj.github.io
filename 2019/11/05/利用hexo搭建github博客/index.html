Hexo是一个快速、简洁且高效的博客框架

安装git和node.js：下载git|node.js

部署到Github

首先在 GitHub 上新建一个空的仓库，名称是「你的用户名.github.io」（注意个用户名是你的GitHub用户名，不是你的电脑用户名）
